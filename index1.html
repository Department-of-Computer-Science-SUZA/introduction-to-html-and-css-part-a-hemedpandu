Java Program Example

Explanation:

This is a simple Java program that calculates the sum of two numbers and displays the result.

Program Code:

public class SumCalculator {
    public static void main(String[] args) {
        int num1 = 5;
        int num2 = 7;
        
        int sum = num1 + num2;
        
        System.out.println("The sum of " + num1 + " and " + num2 + " is: " + sum);
    }
}

Output:

The sum of 5 and 7 is: 12
